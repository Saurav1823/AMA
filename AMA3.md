Q: Difference between undefined and not defined  
A: undefined means a variable is declared but has no value. Not defined means the variable is never declared at all.

Q: What is try-catch block?  
A: It is used to handle errors. try block runs code, if error occurs catch block handles it.

Q: What are shift and unshift?  
A: shift() removes first element from array. unshift() adds element at the beginning of array.

Q: What is splice and its syntax?  
A: splice is used to add/remove elements in array.  
Syntax: array.splice(startIndex, deleteCount, newItems)

Q: What is "this" keyword?  
A: "this" refers to the object that is calling the function.

Q: How to access HTML in JS?  
A: Use DOM methods like document.getElementById(), document.querySelector()

Q: What are types of loops?  
A: for loop, while loop, do-while loop, for...of, for...in

Q: Pre-increment vs Post-increment  
A: Pre (++a): value increases first, then used  
Post (a++): value used first, then increases

Q: Function and variable same name, which is hoisted?  
A: Function declaration is hoisted above variable.

Q: What are template literals?  
A: Strings using backticks (`) that allow variables using ${}.

Q: Difference between null and undefined  
A: null is intentionally empty value. undefined means value is not assigned.

Q: What is closure?  
A: A function that remembers variables from its outer function even after outer function ends.

Q: Difference between find and filter  
A: find() returns first matching element. filter() returns all matching elements in array.

Q: Function declaration vs function expression  
A: Declaration is hoisted. Expression is not hoisted.

Q: What is Object.freeze()?  
A: It makes object immutable cannot change, add, or delete properties.

Q: Which data structure returns element faster?  
A: Objects are faster for lookup than arrays.
