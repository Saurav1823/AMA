Q: How do we get all data to model  
A: We can get all data to a model by querying the database using ORM methods.

Q: What is env  
A: An env (environment file) is used to store configuration values like API keys, database URLs, and secrets. It helps keep sensitive data separate from the main code.  

Q: How to create Django Project  
A: To create a Django project, install Django and run the command django-admin startproject projectname. This creates the project structure with default files.  

Q: Explain stateful and stateless  
A: A stateful system stores user data between requests, like login sessions. A stateless system does not store any user data, and each request is treated as new.  

Q: What is JWT  
A: JWT (JSON Web Token) is a secure token used for authentication. It contains encoded data and is sent between client and server to verify identity without storing sessions.  

Q: How to add class name in DOM  
A: You can add a class name using JavaScript with element.classList.add("classname").  

Q: How do you handle particular element in array in JS  
A: You can handle a specific element using methods like find() to get one item, filter() to get multiple, or by accessing it using its index.  

Q: What is HTTP  
A: HTTP (HyperText Transfer Protocol) is the protocol used for communication between a web browser and a server. It defines how requests and responses are sent.  

Q: What is semantic HTML  
A: Semantic HTML uses meaningful tags like header, footer, and article to clearly describe the structure and content of a webpage.  

Q: What is event delegation  
A: Event delegation is a technique where a parent element handles events for its child elements, improving performance and reducing code.  

Q: What is a header in data packet  
A: A header in a data packet contains important information like source address, destination, and control data needed for proper delivery.  

Q: Explain local storage and session storage  
A: Local storage stores data permanently in the browser until manually cleared, while session storage stores data only for the duration of the browser session.  

Q: What is stack  
A: A stack is a data structure that follows Last In First Out (LIFO), where the last added element is removed first.
