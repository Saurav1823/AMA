### What is !important in CSS?
!important is used to give a CSS rule the highest priority. It overrides other CSS rules even if they have higher specificity.

### What are Inline and Block Elements?
HTML elements are mainly of two types: inline and block. Block elements take the full width and start on a new line, while inline elements take only the required width and stay in the same line.

### What are Inline and Block Elements?
Inline elements do not start on a new line and only take the space they need. Block elements always start on a new line and take the full width available.

### What is Grid in CSS?
CSS Grid is a layout system used to create rows and columns on a web page. It allows developers to design two-dimensional layouts easily.

### What is Inline CSS and Internal CSS?
Inline CSS is written directly inside an HTML element using the style attribute. Internal CSS is written inside a style tag in the head section of an HTML document.

### How to Center Content in the Body?
Content in the body can be centered using CSS properties like flexbox or text alignment so that the content appears in the middle of the page.

### What are div and span?
div is a block element used to group large sections of HTML content. span is an inline element used to style or group small parts of text.

### What are Comments in HTML?
Comments in HTML are used to write notes or explanations in the code that are not displayed on the webpage.

### What is Z-index in CSS?
Z-index controls the vertical stacking order of elements. Elements with a higher z-index appear in front of elements with a lower z-index.

### What is Specificity Order in CSS?
Specificity determines which CSS rule is applied when multiple rules target the same element. The order from lowest to highest priority is: element selector, class selector, ID selector, and !important.

### What is Box Shadow in CSS?
Box shadow is a CSS property used to add shadow effects around an element’s box.

### What is Placeholder in CSS?
Placeholder is the text shown inside an input field to give a hint to the user about what type of information should be entered.

### Does justify-content work on the main axis or horizontal axis?
justify-content works on the main axis of a flex container. By default, the main axis is horizontal, so it usually aligns items horizontally.

### What are Anchor Tags?
Anchor tags are used in HTML to create hyperlinks that allow users to navigate from one page to another.

### How to Link an HTML Page to an Element?
An HTML page can be linked to a specific element using an anchor link with an element ID so that clicking the link takes the user directly to that section of the page.
